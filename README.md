# 「一都三県（市区町村別）空き家率マップ」の作成

## 全体構成（IMRAD形式）
### Abstract
本研究は、一都三県(東京都・神奈川県・千葉県・埼玉県）の市区町村単位の空き家率を総務省統計局の住宅・土地統計調査より個別に算出し、そのデータをもとにしてデジタルマップ上に視覚化したものである。
